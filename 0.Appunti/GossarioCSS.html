<border> il border e' il bordo di ogni elemento e viene definito con tre proprieta'. lo spessore = border-width, lo stile = border-style ed il colore = border-color.
codice: border: 2px dotted blue

background Consente di definire lo sfondo di un elemento ed e' totalmente modificabile con colori e immagini.
calc() calc() e' una funzione CSS che ti permette di definire una misura in base ad altre misure sulla pagina, cioe' puoi dinamicizzare la misura di un elemento figlio in base alla misura dell'elemento padre.
codice: td{background-color: yellow; width: 100px;} div{background-color: blue; width: calc(100% - 30px);} 		width: 100px;
width: calc(100% - 30px)
cursor cursor permette di modificare la freccetta del mouse.

content Usando ::before o ::after dopo l'elemento prescelto,(es: div::before) si aggiunge l'icona desiderata prima o dopo per l'elemento opportunamente identificato.
bella questa icona!
::before / ::after Sono pseudo elementi, non sono presenti nella pagina. Si usano principalmente per aggiungere contenuto ad un elemento (con caratteristiche specifiche) ripetuto nella pagina un carattere speciale o un icona. vedi content
display Serve per gestire come un elemento viene mostrato a video e se viene mostrato. le proprieta' principali che lo riguardano sono: block, in-line, none, flex, grid ...
:first-child La pseudo-classe CSS :first-child rappresenta il primo elemento tra un gruppo di elementi fratelli. es: abbiamo una lista di tre elementi e nel CSS vado a specificare il bordo solo per il primo elemento in questo modo: li:first-child{ border: 1px solid red; }

:hover La pseudo-classe CSS :hover corrisponde quando l'utente interagisce con un elemento con un dispositivo di puntamento, ma non lo attiva necessariamente. Generalmente viene attivato quando l'utente passa sopra un elemento con il cursore (puntatore del mouse).
codice: .container:hover{ background-color: brown; } 		container
