<head> L'elemento HTML <head> contiene informazioni leggibili dalla macchina (metadati) sul documento, come <title>, <stript> e <style>. ...
<link> Specifica le relazioni tra il documento corrente e una risorsa esterna. Questo elemento è più comunemente usato per collegarsi ai CSS, ma è anche usato per stabilire le icone del sito (sia icone in stile "favicon" che icone per la schermata iniziale e le app sui dispositivi mobili) tra le altre cose. ...
<body> L'elemento HTML <body> rappresenta il contenuto di un documento HTML. Può esserci un solo elemento <body> in un documento ...
<a> L'elemento HTML <a> (o elemento anchor), con il suo attributo href, crea un collegamento ipertestuale a pagine Web, file, indirizzi e-mail, posizioni nella stessa pagina o qualsiasi altra cosa che un URL può indirizzare. Il contenuto all'interno di ogni <a> deve indicare la destinazione del collegamento. Se l'attributo href è presente, premendo il tasto Invio mentre ci si concentra sull'elemento <a> lo si attiverà. <main> L'elemento HTML <main> rappresenta il contenuto dominante del <body> di un documento. L'area del contenuto principale è costituita da contenuto direttamente correlato o che si espande sull'argomento centrale di un documento o sulla funzionalità centrale di un'applicazione. ...
<blockquote> L'elemento HTML <blockquote> indica che il testo racchiuso è una citazione estesa. Di solito, questo viene reso visivamente mediante indentazione. Un URL per la fonte della citazione può essere dato usando l'attributo cite, mentre una rappresentazione testuale della fonte può essere data usando l'elemento <cite>.

<cite> L'elemento HTML <cite> viene utilizzato per contrassegnare il titolo di un'opera creativa citata. Il riferimento può essere in forma abbreviata secondo le convenzioni appropriate al contesto relative ai metadati di citazione. ...
<div> L'elemento HTML <div> è un contenitore generico. Non ha alcun effetto sul contenuto o sul layout fino a quando non viene stilizzato in qualche modo utilizzando i CSS (ad esempio, lo stile viene applicato direttamente ad esso o un qualche tipo di modello di layout come Flexbox viene applicato al suo elemento genitore). ...
<i> L'elemento HTML <i> rappresenta un intervallo di testo che è separato dal testo normale per qualche motivo, come testo idiomatico, termini tecnici, designazioni tassonomiche, tra gli altri. <strong> L'elemento HTML <strong> indica che i suoi contenuti hanno una forte importanza, serietà o urgenza. I browser in genere visualizzano i contenuti in grassetto. il testo in grassetto e' un elemento <strong>
<img> L'elemento HTML <img> incorpora un'immagine nel documento. Si utilizza l'attributo "src" per includere il link dell'immagine.Si utilizza l'attributo "alt" per includere una descrizione testuale nel caso l'immagine non si caricasse correttamente.
